WHAT TIME IS IT?

INTRODUCTION



WHAT TIME IS IT? is a website where children/students can learn how to read the time accurately and properly, this website does not only show tutorials but it also provides opportunities where children/students can interact with different types of challenges in the quiz section. The aim of this website is to teach children/students to learn more effectively as it have both visual and touch interactivity. Hopefully over the use of this website, users can learn something new and get even more motivated to learn.
